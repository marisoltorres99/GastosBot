# GastoBot

Bot de Telegram para gestión de gastos personales y vencimientos.

## 🚀 Tecnologías

* Python
* Flask
* SQLAlchemy
* MySQL
* Docker
* APScheduler
* Flask-Migrate

## ✨ Funcionalidades

* Registro de gastos
* Categorías personalizadas
* Resumen mensual
* Gestión de vencimientos
* Recordatorios automáticos
* Eliminación de registros

## ⚙️ Instalación

```bash
pip install -r requirements.txt
```

## ▶️ Ejecutar proyecto

```bash
python app.py
```

## 🗄️ Migraciones

```bash
flask db migrate -m "mensaje"
flask db upgrade
```

## 📌 Estado

Proyecto personal en desarrollo orientado a práctica de backend, automatización y arquitectura de aplicaciones con Python.
